## Features
- Game Play
- Game Sound (Hitting objects)
- Game Scoring
	- Cyan: +10 score
	- White: +30 and adjust the game area
	- Red: Reduce Game area
- Viewing of all scores

## Tools/Tech/Libraries
- Java
- Android Studio 3.0.1
- ButterKnife
- Room Persistance Library
- RxJava
- RxAndroid
- SlidingRootNav
- TableView
